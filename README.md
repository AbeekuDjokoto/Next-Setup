### Setup

1. Clone the repo

   ```sh
   git clone -b dev --single-branch https://github.com/Wookin-Inc/ownkey-real-estate-main-web.git
   ```

2. Install Packages

   ```sh
   npm install
   ```

3. Compile and Hot-Reload for Development

   ```sh
   npm run dev
   ```
